# Декораторы логирования — Учебный проект Нетологии

Этот проект создан в рамках учебного курса по Python в [Нетологии](https://netology.ru/).  
Цель — реализация двух декораторов для логирования вызовов функций:

- `logger_1` — записывает логи в файл `main.log` без передачи аргументов.
- `logger_2(path)` — записывает логи в указанный файл.

## Функционал

- Логирование:
  - Имени функции
  - Позиционных и именованных аргументов
  - Результата выполнения
  - Времени вызова (в формате `ГГГГ-ММ-ДД ЧЧ:ММ:СС`)
- Обработка ошибок записи в файл
- Человеко-читаемый формат логов

## Тесты

В проекте реализованы автоматические тесты:
- `test_1()` — проверяет `logger_1`
- `test_2()` — проверяет `logger_2` на нескольких файлах

Тесты проверяют:
- Создание файла лога
- Корректность записи данных
- Соответствие формата

## Примечание

Учебный проект выполнен в рамках курса Нетологии по Python-разработке.
